## Instalação

Você poderá configurar tudo usando apenas o windows cli. 
É importante ter instalado o gcc previamente: <a href="https://www.mingw-w64.org/">gcc</a> 
## Aqui estão os comandandos de configuração a serem seguidos.
```bash
cd desktop/

```

```bash
git clone https://github.com/Deividev365/calculadora.git

```

```bash
cd calculadora

```

```bash
gcc -o teste app.c

```

```bash

.\teste.exe

```
